=== PROBLEM STATEMENTS (for copy-paste) ===

Two Sum:
Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

Fibonacci:
Write a function that returns the nth Fibonacci number. F(n) = F(n-1) + F(n-2) for n > 1.

Palindrome:
Determine if a given string is a palindrome, ignoring spaces and case.

String Reversal:
Reverse a given string using various approaches and compare performance.

Binary Search:
Implement binary search to find the index of a target value in a sorted array. Return -1 if not found.

Bubble Sort:
Implement the bubble sort algorithm to sort an array of integers.

Linked List:
Implement a singly linked list with insert, delete, and search operations.

Stack:
Implement a stack data structure with push, pop, peek, and helper methods.

Async Fetch:
Implement async API data fetching with error handling, retries, and caching.

Concurrent Sum:
Calculate array sum using concurrent goroutines with proper synchronization.
